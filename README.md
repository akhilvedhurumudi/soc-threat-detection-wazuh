# soc-threat-detection-wazuh
# SOC Threat Detection System using Wazuh

This project demonstrates a Security Operations Center monitoring system.

## Tools Used
- Wazuh
- Sysmon
- Linux Server
- MITRE ATT&CK framework

## Features
- File Integrity Monitoring
- Brute-force attack detection
- Security event monitoring
- Automated response actions

## Architecture
Endpoint Agents → Wazuh Manager → Security Dashboard

## Results
The system detects suspicious activity and generates alerts for security analysts.

## Conclusion
This project demonstrates real-time threat detection using a SIEM platform.
